Guia Completo de Fontes no CSS

1. O que é a propriedade font-family?

A propriedade font-family no CSS é usada para especificar a fonte de um elemento. Você pode listar várias fontes em ordem de preferência (chamada de "pilha de fontes").

seletor {
  font-family: "Nome da Fonte", familia-generica, fallback;
}

2. Exemplos Práticos

Fonte padrão do sistema:

Este texto está usando Arial, Helvetica, ou uma fonte sans-serif genérica como fallback.

font-family: Arial, Helvetica, sans-serif;

Fonte do Google Fonts (Poppins):

Este texto está usando a fonte Poppins importada do Google Fonts.

@import url('https://fonts.googleapis.com/css2?family=Poppins&display=swap');
font-family: 'Poppins', sans-serif;

Fonte serifada segura para web:

Este texto está usando Georgia, Times New Roman, ou uma fonte serif genérica.

font-family: 'Georgia', 'Times New Roman', serif;

4. Fontes Web-Safe (Seguras para Web)

Estas fontes são instaladas na maioria dos dispositivos:

  • Sans-serif: Arial, Helvetica, Verdana, Tahoma, Trebuchet MS
  • Serif: Georgia, Times, "Times New Roman"
  • Monospace: "Courier New", Courier, monospace

5. Fontes Genéricas (Fallback)

Quando nenhuma fonte específica está disponível, use famílias genéricas:

  • serif - Fontes com serifas (pequenos traços nas extremidades)
  • sans-serif - Fontes sem serifas
  • monospace - Fontes de largura fixa
  • cursive - Fontes que imitam escrita à mão
  • fantasy - Fontes decorativas

Recursos para encontrar fontes:

  • Google Fonts: https://fonts.google.com/ (gratuito)
  • Adobe Fonts: https://fonts.adobe.com/ (assinatura)
  • Font Squirrel: https://www.fontsquirrel.com/ (gratuito)
  • DaFont: https://www.dafont.com/ (gratuito)

6. Como importar fontes do Google Fonts

Passo a passo para usar fontes externas:

  1. Visite https://fonts.google.com/
  2. Escolha uma fonte e clique nela
  3. Selecione os estilos desejados (normal, negrito, etc.)
  4. Copie o código de importação e cole no topo do seu CSS
  5. Use o nome da fonte na propriedade font-family
/* No CSS */
@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

body {
  font-family: 'Roboto', sans-serif;
}
